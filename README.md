## Overview

`PSASS_plot` is an R-based visualization script designed to generate **publication-ready genomic plots** from PSASS outputs.

It takes the standard PSASS window file:

```
psass_window.tsv
```

and produces:

*  Multi-panel Manhattan plot
*  Circos plot

These plots highlight key genomic signals:

* **Fst (genetic differentiation)**
* **Sex-specific SNP density**
* **Depth ratio variation**

---

## Key Features

* ✔ Clean, publication-quality figures
* ✔ Automatic contig ordering
* ✔ Single-chromosome zoom mode with Mb/kb scaling
* ✔ Region-specific plotting
* ✔ Optional filled Manhattan plots (`--p`)
* ✔ Works directly on raw PSASS output (no preprocessing needed)

## Usage

Run inside the directory containing `psass_window.tsv`:

```bash
Rscript psass_plot.R
```

---

## Parameters

### `--n`

Plot only the first **N contigs**:

```bash
Rscript psass_plot.R --n 25
```

---

### `--chr`

Plot a **single contig** (enables genomic coordinate axis in Mb/kb):

```bash
Rscript psass_plot.R --chr 11
```

⚠️ Important:

* Uses **index order**, not chromosome name

---

### `--region`

Zoom into a specific region (requires `--chr`):

```bash
Rscript psass_plot.R --chr 11 --region 1:100000
```

---

### `--p`

Enable filled Manhattan plots:

```bash
Rscript psass_plot.R --p
```

---

## Summary

| Command                | Output behavior                |
| ---------------------- | ------------------------------ |
| *(no args)*            | All contigs (genome-wide view) |
| `--n X`                | First X contigs                |
| `--chr X`              | Single contig (Mb-scaled axis) |
| `--chr X --region A:B` | Zoomed region                  |
| `--p`                  | Filled Manhattan plots         |

---

## Important Notes

* `--chr` refers to **ordered contig index**, not name
* `--region` only works with `--chr`
* `--n` and `--chr` are mutually exclusive
* If only one contig exists, genome-wide mode may still behave differently than `--chr`

## Example Workflow

```bash
# Basic genome-wide plot
Rscript psass_plot.R

# Focus on chromosome 2
Rscript psass_plot.R --chr 2

# Zoom into region
Rscript psass_plot.R --chr 2 --region 50000:350000

# Publication-style filled plot
Rscript psass_plot.R --chr 2 --p
```
